# Slipstream

Native screen and camera recorder with built-in transcript editing, captions, cinematic zoom, and vertical reel export for Windows, macOS, and Linux.

Slipstream is built for creators who want to record, edit, and ship short-form demos without uploading footage to a cloud editor. Recording, editing, transcription, captions, camera layout, and export are designed to happen on-device.

## Features

- Screen and camera recording for desktop workflows
- Transcript-based editing for quick cuts
- Cinematic cursor zooms for demos and walkthroughs
- Camera picture-in-picture and split layouts
- Captions for short-form clips
- One-click 9:16 reel export
- Free desktop builds for Windows, macOS, and Linux
